===Windows Win32 version===
Run the .exe file and play

===Windows UWP version===
Go to settings, enable Developer Mode to install the package.

Then, install all dependencies in the Dependencies folder, and then install the MonoSpace.msix file to install the game itself.

===Linux version===
Right click the MS.x86_64 file, set it as executable by right clicking it and going into properties.
Then, double click to play.
